Step 1 · AI chat

Video details → voice-over script prompt

Fill in the new video's details. The app builds the complete, ready-to-paste prompt (with your sample scripts and target duration already inside).

Your generated prompt

What to do with it

  1. Paste the prompt into Claude / ChatGPT / any AI chat.
  2. It goes straight to building the file (no script text printed in the chat) and should hand you a downloadable voice-over-script.txt file directly — save it into the video's folder (the folder with build_timeline.py). If your AI tool can't create files, it falls back to printing the script as chat text — copy that instead.
  3. Open the downloaded file to read it — ask for changes in the same chat and re-download until you like it.
New video = new folder: copy build_timeline.py, make_flow_prompts.py, Rename-Scenes.ps1 into it and create an empty Scene folder (see the run sheet in Reference).
Step 2 · AI chat

Voice-over script → scene prompts

Paste the text of your finished voice-over-script.txt below. Every two consecutive breath-lines become one image scene, roughly halving the number of images you need to generate. The app checks it and builds the full scene-prompts prompt for you.

Your generated prompt

What to do with it

  1. Paste the prompt into the AI chat.
  2. It first answers TOTAL SCENES: N — check N matches the scenes / images count shown above (about half your line count).
  3. It then goes straight to building the file (no scene list printed in the chat) — for a long script this may take a while to finish.
  4. It should hand you a downloadable original-scenes-prompts.txt file directly — save it into the video's folder. If your AI tool can't create files, it falls back to printing all scenes as chat text instead — copy that and save it under that name.
Step 3 · Command

Make the Flow prompt list

One command turns original-scenes-prompts.txt into the Flow-ready prompt list — and verifies every scene sentence against your script, line by line.

Run in the video's folder

py make_flow_prompts.py

Open a terminal in the video's folder (in File Explorer: click the address bar, type cmd, press Enter) and paste the command.

What you get

  1. flow-ai-scene-prompts.txt — one image prompt per scene, separated by blank lines.
  2. A verification result: "Verified: all N scene sentences match voice-over-script.txt" means you're safe.
If it prints WARNINGS — the AI merged, skipped or reworded a line. Fix those scenes in original-scenes-prompts.txt (or ask the AI chat to redo them) and run the command again before generating any images.
Step 4 · Reminder — your bulk extension

Generate the images on Flow AI

You already have this covered with your third-party bulk extension — this step is just the checklist so nothing is missed.

  1. Open Flow AI and load your character reference image (every prompt expects it).
  2. Feed your bulk extension the prompts from flow-ai-scene-prompts.txt — keep the order.
  3. Generate, then download all images into the Scene folder.
Downloaded names like 1.png, 1 (1).jpg, 2.webp are perfect — the renamer groups them by that leading number. Multiple variants of the same scene are fine (they become A / B / C).
Step 5 · Command

Rename the images

The renamer reads the scene sentences from original-scenes-prompts.txt and renames every image to 001 - sentence.jpg style.

Run in the video's folder

powershell -ExecutionPolicy Bypass -File "Rename-Scenes.ps1"

Or simply right-click Rename-Scenes.ps1 → Run with PowerShell.

It prints Scenes / Groups / Images counts — they should match your scene count. If it warns that counts differ, a scene is missing an image (or has a stray extra) — fix that before continuing.
Step 6 · Recording

Record the voice-over

Read voice-over-script.txt out loud — it's formatted as one breath per line, so pause naturally at each line break.

  1. Record at your natural pace — the timeline adapts to your timing automatically, so don't rush.
  2. Small stumbles are fine if you re-read the line cleanly; big retakes are better cut from the audio before the next step.
  3. (Optional) save the final audio as narration.wav in the video's folder — then Step 8 can place it in the Premiere sequence automatically.
Step 7 · Reminder — Cutback extension

Create Transcript.srt

You already do this with the Cutback extension — just the checklist:

  1. Run your recorded audio through Cutback to get the subtitles.
  2. Export/save as Transcript.srt in the video's folder.
Short subtitle lines, ALL-CAPS, digits instead of words — all fine. The timeline builder normalizes everything ("8" matches "eight", etc.).
Step 8 · Command

Build the timeline

One command matches every image to the exact seconds it is spoken, adds the zoom/slide motion and the 5-frame dissolves, and writes timeline.xml.

Run in the video's folder

py build_timeline.py --scale 72

Check the result

  1. The console prints every scene's match — look at the summary line: similarity around 0.9+ is great.
  2. Look for LOW CONFIDENCE flags in the console output — those few scenes (if any) are the only cuts worth eyeballing in Premiere.
Motion is included automatically: slow zoom in/out on every image, a slide every 5–8 clips (zoom direction flips after each slide), and a 5-frame cross dissolve at every cut — none at the very start of image 1.
Step 9 · Premiere Pro

Import into Premiere

The last step — everything arrives already assembled.

  1. In Premiere: File → Import → pick timeline.xml.
  2. Open the imported sequence — images, timing, motion and transitions are already in place.
  3. If you didn't use --audio, drop your narration on A1 at 0:00.
  4. Play it through once; nudge any LOW CONFIDENCE cut from the build output if needed.
🎉 That's the whole pipeline. Hit ☁ Save video one last time, then Start a new video in the sidebar when you begin the next one.

Quick run sheet — new video, start to finish

  1. New folder → copy build_timeline.py, make_flow_prompts.py, Rename-Scenes.ps1 → create an empty Scene folder.
  2. App Step 1: title + key points → copy prompt → AI → save as voice-over-script.txt.
  3. App Step 2: paste the script → copy prompt → AI, reply continue → save all output as original-scenes-prompts.txt.
  4. Run py make_flow_prompts.py → must print Verified.
  5. Flow AI: character reference + bulk extension with flow-ai-scene-prompts.txt → download all images into Scene.
  6. Run Rename-Scenes.ps1 → counts must match.
  7. Record the voice-over reading voice-over-script.txt (optional: save narration.wav).
  8. Cutback → save as Transcript.srt.
  9. Run py build_timeline.py --scale 72 (add --audio narration.wav if saved) → check the console for LOW CONFIDENCE.
  10. Premiere → File → Import → timeline.xml. 🎉
